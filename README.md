# 🏥 AI-Powered Home Visit Doctor Appointment Booking System

An AI-powered full-stack healthcare platform for booking home-visit doctor appointments. The platform includes an AI-based Symptom Checker and Multilingual Support for patients, along with an AI-assisted Automatic Prescription Generator for doctors, making healthcare more accessible, efficient, and intelligent.

---

## 📌 Features

### 👤 Patient Panel
- User Registration & Login
- Book Home Visit Doctor Appointments
- Search Doctors by Specialization
- View Appointment History
- Profile Management
- AI Symptom Checker
- Multi-language Support

---

### 👨‍⚕️ Doctor Panel
- Doctor Login
- Manage Profile
- View Assigned Home Visits
- Accept/Reject Appointments
- Update Appointment Status
- Review & Approve AI-generated Prescriptions

---

### 👨‍💼 Admin Panel
- Admin Dashboard
- Add & Manage Doctors
- View All Appointments
- Monitor Appointments

---

# 🤖 AI Features

- AI-based Symptom Checker
- AI-generated Draft Prescriptions
- Multi-language and Speech-to-Text Support
- Google Gemini AI Integration

---

# ⚙️ Installation

## Clone Repository

---

## Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Admin

```bash
cd admin
npm install
```

### Backend

```bash
cd backend
npm install
```

---

# ▶️ Run the Project

### Backend

```bash
npm start
```

### Frontend

```bash
npm run dev
```

### Admin

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **backend** folder.

```env
MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

CLOUDINARY_NAME=your_cloudinary_name
CLOUDINARY_API_KEY=your_cloudinary_api_key
CLOUDINARY_SECRET=your_cloudinary_secret

GEMINI_API_KEY=your_google_gemini_api_key

```
